# Icon

## Installation

### Using packaging manager

```shell
# Choose a package manager you like.

# NPM
$ npm install @echone-ui/icons
# Yarn
$ yarn add @echone-ui/icons
# pnpm
$ pnpm install @echone-ui/icons
```
