# Button 按钮

## 按钮类型

你可以使用 variant 属性来定义按钮类型。<br> 包括default、secondary、destructive、ghost、outline、link.

:::demo

button/basic

:::

## 禁用

你可以使用 disabled 属性来定义按钮是否被禁用。使用 disabled 属性来控制按钮是否为禁用状态。 <br> 该属性接受一个Boolean 类型的值。

:::demo

button/disabled

:::

## 图标按钮

使用图标为按钮添加更多的含义。

:::demo

button/icon

:::

## 加载状态按钮

点击按钮来加载数据，并向用户反馈加载状态。

:::demo

button/loading

:::
